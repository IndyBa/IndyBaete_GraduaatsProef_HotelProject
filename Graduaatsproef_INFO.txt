INFO BACKEND:
De backend is ontwikkeld in Java met behulp van het IntelliJ IDE. Het project is opgezet met het Spring Boot Framework en enkele andere dependencies die je kunt vinden in het pom.xml-bestand van het backendproject. Om de backend te koppelen aan de database, kan het nodig zijn om het application.properties bestand te bewerken. Dit bestand bevindt zich in de map: hotelproject\backend\src\main\resources\application.properties 
Mogelijk moet je de velden voor url, user en password aanpassen aan de waarden die overeenkomen met jouw MySQL-database. Zorg er vervolgens voor dat je het project uitvoert via de HotelProjectApplication klasse om de backend te doen werken.
